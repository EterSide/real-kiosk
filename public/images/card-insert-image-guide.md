# 카드 삽입 이미지 저장 안내

## 📍 이미지 저장 위치
제공하신 카드 삽입 이미지를 다음 경로에 저장해주세요:

```
public/images/card-insert.png
```

## 📝 방법

### 1. 이미지 준비
- 제공하신 "Insert Card Here" 이미지를 다운로드하거나 저장합니다.
- 파일명을 `card-insert.png`로 변경합니다.

### 2. 저장 위치
```
real-kiosk/
└── public/
    └── images/
        └── card-insert.png  ← 여기에 저장!
```

### 3. 확인
- 브라우저에서 `http://localhost:3000/images/card-insert.png`로 접근해서 이미지가 보이는지 확인합니다.

## 💡 대체 방법

이미지가 없어도 결제 시스템은 정상 작동합니다:
- 이미지 로드 실패 시 자동으로 텍스트 안내로 대체됩니다.
- "💳 카드를 투입구에 넣어주세요 →" 텍스트가 표시됩니다.

## ✅ 완료 후
이미지를 저장한 후 개발 서버를 새로고침하면 결제 화면에서 이미지가 표시됩니다.
